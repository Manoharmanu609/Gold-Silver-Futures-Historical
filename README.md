1️ Project Title

Gold and Silver Futures Price Analysis & Prediction Using Python

2️ Introduction

Gold and Silver are two important precious metals used for investment, jewelry, and industrial purposes. Their prices change based on factors like inflation, global demand, interest rates, and economic conditions.

This project analyzes historical Gold and Silver futures data using Python to understand trends, volatility, and future price prediction.Comparative Analysis and Prediction of Gold and Silver Futures Prices


3️ Objectives

To analyze historical Gold and Silver price data

To compare price trends of both metals

To measure volatility and returns

To build a simple prediction model using Machine Learning

To visualize insights using graphs

6️ Methodology
Step 1: Data Cleaning

Converted Date column into datetime format

Removed commas from Price column

Sorted data by date

Checked for missing values

Step 2: Exploratory Data Analysis (EDA)

Line charts to analyze trends

Bar charts for monthly/yearly averages

Box plots to detect volatility

Daily returns calculation

Step 3: Comparison of Gold and Silver

Merged datasets using Date

Created comparison line chart

Normalized data for fair performance comparison

Step 4: Machine Learning Prediction

Used Linear Regression model

Converted Date into numeric format

Split dataset into training and testing

Evaluated model using Mean Squared Error (MSE) and R² Score

7️ Results

Both Gold and Silver show long-term upward trends.

Silver is more volatile than Gold.

Gold provides stable growth, suitable for long-term investment.

Both metals are positively correlated (move in the same direction).

8️ Conclusion

This project successfully analyzed and compared Gold and Silver futures prices.

Gold is more stable and safer for investment, while Silver shows higher fluctuations but potential short-term gains.

Using Python and machine learning techniques, we gained insights into price behavior and built a basic prediction model.

Future improvements can include advanced time-series models like ARIMA or LSTM for better prediction accuracy.
